# Exemplo — Planejamento para Biblioteca Python

## Pedido do usuário

Crie uma coleção de estudos sobre a biblioteca Python Rich.

## Resposta esperada da IA

Antes de gerar o PDF, proponho o seguinte plano.

Tema: biblioteca Python Rich.

Objetivo: ensinar a usar a biblioteca para criar saídas ricas no terminal, com textos formatados, tabelas, painéis, barras de progresso, logs e interfaces de terminal mais legíveis.

Sugestão de coleção:

- Introdução Geral — O que é Rich e por que estudar.
- Volume 0 — Ambiente, mentalidade de terminal e primeiros exemplos.
- Volume 1 — Textos, cores, estilos e impressão formatada.
- Volume 2 — Tabelas, painéis, markdown e layout.
- Volume 3 — Progress bars, logs, status e aplicações práticas.
- Volume 4 — Projeto prático, boas práticas, erros comuns e exercícios.

Cada volume pode ficar em torno de 50 páginas.

Perguntas:

1. Quer cobrir todas as funções principais ou apenas as mais usadas?
2. Qual nível deseja?
3. Quer exemplos linha por linha?
4. Quer relacionar com algum projeto seu?
5. Qual formatação prefere?

## Variação — Pedido avançado

Se o usuário pedir uma biblioteca Python X em conteúdo avançado, a IA deve usar o exemplo `exemplo_biblioteca_python_avancada.md` como referência.

## Variação — Formação de especialista

Se o usuário pedir para se tornar especialista na biblioteca Python X, com todas as funções e possibilidades, a IA deve usar o exemplo `exemplo_biblioteca_python_especialista.md` como referência.

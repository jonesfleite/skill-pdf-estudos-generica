# Modelo — Framework Web

## Quando usar

Use quando o tema for um framework web, backend, frontend ou full-stack.

## Perguntas específicas

1. O foco é iniciante, intermediário ou profissional?
2. O material deve cobrir estrutura de projeto?
3. Deve incluir banco de dados?
4. Deve incluir autenticação?
5. Deve incluir testes?
6. Deve incluir deploy?
7. Quer relacionar com algum projeto seu?

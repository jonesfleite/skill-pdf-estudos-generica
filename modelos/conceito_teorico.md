# Modelo — Conceito Teórico

## Quando usar

Use quando o tema for uma ideia, teoria, princípio, método ou conceito.

## Perguntas específicas

1. O objetivo é entender, aplicar ou revisar?
2. Quer exemplos simples?
3. Quer exemplos profissionais?
4. Quer exercícios?
5. Quer relacionar com algum projeto seu?

## Subassuntos recomendados

- Definição.
- Origem ou contexto.
- Problema que resolve.
- Ideias principais.
- Analogias.
- Exemplos.
- Comparações.
- Aplicações.
- Limitações.
- Erros de interpretação.
- Exercícios.

# Modelo — Linguagem de Programação

## Quando usar

Use quando o tema for uma linguagem de programação.

## Perguntas específicas

1. O material deve ir do zero ao avançado?
2. Deve incluir orientação a objetos?
3. Deve incluir bibliotecas padrão?
4. Deve incluir testes?
5. Deve incluir projetos práticos?
6. Quer relacionar com algum projeto seu?

## Subassuntos recomendados

- História e finalidade.
- Instalação.
- Ambiente.
- Sintaxe.
- Tipos.
- Variáveis.
- Controle de fluxo.
- Funções.
- Módulos.
- Erros e exceções.
- Arquivos.
- Programação orientada a objetos.
- Testes.
- Boas práticas.
- Projetos.

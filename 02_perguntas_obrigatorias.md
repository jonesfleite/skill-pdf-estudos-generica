# 02 — Perguntas Obrigatórias

## Objetivo

Garantir que a IA entenda o escopo antes de gerar o PDF ou coleção.

## Regra principal

Quando o pedido for amplo, a IA deve fazer perguntas antes de criar o PDF final.

A IA deve evitar fazer uma lista enorme de perguntas sem necessidade. Deve propor um plano inicial e perguntar o essencial.

## Perguntas mínimas

Quando o usuário pedir um PDF ou coleção, perguntar:

1. Qual nível deseja?
   - iniciante;
   - intermediário;
   - avançado;
   - profissional;
   - do zero ao avançado.

2. Quer PDF único ou coleção em volumes?

3. Se for coleção, a IA pode definir a quantidade de volumes conforme o assunto?

4. Cada volume pode ter aproximadamente 50 páginas?

5. Quais subassuntos devem entrar?

6. Quais subassuntos devem ficar de fora?

7. O material deve ter exemplos práticos?

8. O material deve ter código, comandos, fórmulas, tabelas ou diagramas?

9. Se houver código ou comandos, deve explicar linha por linha?

10. Quer exercícios ao final dos capítulos ou volumes?

11. Quer um projeto prático ou estudo de caso no final?

12. Quer relacionar o assunto com algum projeto seu?

13. Qual formatação prefere?
    - livro didático confortável;
    - apostila técnica;
    - leitura em tablet;
    - impressão econômica.

## Pergunta obrigatória sobre projeto do usuário

Sempre perguntar:

"Você quer que o material relacione este assunto com algum projeto seu? Se sim, descreva brevemente o projeto, o objetivo e onde esse assunto deve ser aplicado."

## Quando o usuário não quiser responder muito

Se o usuário disser para a IA decidir, assumir:

- nível: do básico ao intermediário;
- formato: coleção se o tema for amplo, PDF único se o tema for pequeno;
- volumes: quantidade definida pela complexidade do assunto;
- tamanho: cerca de 50 páginas por volume;
- estilo: livro didático confortável;
- exemplos: sim;
- exercícios: sim;
- código: sim quando o tema for programação ou tecnologia;
- projeto do usuário: não relacionar, salvo se o usuário informar um projeto.

## Forma correta de perguntar

A IA deve primeiro sugerir o escopo e depois perguntar.

Exemplo:

"Para este tema, sugiro uma coleção com 4 volumes. Os subassuntos principais seriam fundamentos, recursos essenciais, exemplos práticos e projeto final. Quer incluir, remover ou alterar algum subassunto? Também quer relacionar o conteúdo com algum projeto seu?"

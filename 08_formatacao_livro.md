# 08 — Formatação de Livro

## Objetivo

Definir como a IA deve perguntar e aplicar a formatação do PDF.

A formatação deve favorecer leitura confortável em tablet, computador e impressão.

## Regra principal

Antes de gerar o PDF final, a IA deve perguntar qual padrão de formatação o usuário prefere.

Se o usuário não escolher, usar o padrão recomendado de livro didático confortável.

## Opções de formatação

### Opção 1 — Livro didático confortável

Usar como padrão.

Características:

- fonte adequada para leitura longa;
- tamanho de letra confortável;
- margens equilibradas;
- espaçamento agradável;
- títulos bem destacados;
- capítulos iniciando em nova página;
- blocos de código legíveis;
- visual limpo.

### Opção 2 — Apostila técnica

Indicada para conteúdo com muitos exemplos, comandos, tabelas e código.

Características:

- fonte muito legível;
- títulos objetivos;
- blocos de código com bom espaçamento;
- tabelas organizadas;
- destaque para observações importantes;
- diagramação prática.

### Opção 3 — Leitura em tablet

Indicada para leitura em tela.

Características:

- fonte maior;
- margens moderadas;
- parágrafos confortáveis;
- menos conteúdo por página;
- títulos bem espaçados;
- código com tamanho adequado.

### Opção 4 — Impressão econômica

Indicada para impressão.

Características:

- margens menores;
- fonte legível e compacta;
- menos páginas;
- uso moderado de espaços;
- visual limpo.

## Pergunta obrigatória

Perguntar:

"Qual formatação você prefere: livro didático confortável, apostila técnica, leitura em tablet ou impressão econômica?"

## Padrão automático

Se o usuário disser para a IA decidir, usar:

- formato: livro didático confortável;
- texto: legível para leitura longa;
- títulos: bem destacados;
- código: monoespaçado e legível;
- margens: confortáveis;
- espaçamento: adequado;
- capítulos: iniciando em nova página.

## Regra de silêncio sobre formatação

A formatação deve ser aplicada, mas não deve ser explicada dentro do PDF.

Não incluir no conteúdo frases sobre:

- fonte usada;
- tamanho da letra;
- margens;
- diagramação;
- escolha visual;
- configuração do PDF.

O PDF deve conter apenas o assunto estudado.

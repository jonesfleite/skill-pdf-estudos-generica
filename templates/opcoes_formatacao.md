# Template — Opções de Formatação

Escolha uma opção de formatação:

1. Livro didático confortável — melhor opção padrão para leitura longa.
2. Apostila técnica — melhor para muitos códigos, comandos e tabelas.
3. Leitura em tablet — fonte maior e diagramação confortável para tela.
4. Impressão econômica — menos páginas e visual mais compacto.

Se não houver escolha, usar livro didático confortável.

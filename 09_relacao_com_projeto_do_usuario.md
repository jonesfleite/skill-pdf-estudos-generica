# 09 — Relação com Projeto do Usuário

## Objetivo

Permitir que a skill seja genérica para qualquer pessoa, mas adaptável a projetos específicos quando o usuário desejar.

## Regra principal

A IA deve perguntar se o usuário quer relacionar o assunto com algum projeto próprio.

A skill não deve mencionar projetos específicos por padrão.

Somente relacionar o conteúdo a um projeto se o usuário informar o projeto ou pedir essa adaptação.

## Pergunta obrigatória

Durante o planejamento, perguntar:

"Você quer que o material relacione este assunto com algum projeto seu? Se sim, descreva brevemente o projeto, o objetivo, o público e onde esse assunto será aplicado."

## Informações úteis sobre o projeto

Se o usuário quiser relacionar o tema a um projeto, pedir ou inferir:

- nome do projeto, se existir;
- objetivo do projeto;
- público-alvo;
- área de aplicação;
- tecnologias usadas;
- problema que o projeto resolve;
- onde o assunto estudado entra;
- nível de detalhe desejado;
- se o projeto deve aparecer em exemplos, exercícios ou estudo de caso.

## Como adaptar o PDF ao projeto

Quando o usuário informar um projeto, a IA pode incluir:

- exemplos contextualizados;
- exercícios aplicados;
- estudo de caso;
- arquitetura de uso;
- capítulo aplicado;
- projeto prático final;
- perguntas de reflexão;
- checklist de aplicação.

## Onde colocar a relação com o projeto

A relação com o projeto pode aparecer em:

- exemplos práticos;
- exercícios;
- capítulo aplicado;
- projeto final;
- estudo de caso;
- conclusão;
- próximos passos.

## Regra de moderação de uso

Não transformar todo o PDF em propaganda ou descrição do projeto.

O assunto principal continua sendo o tema de estudo.

O projeto do usuário deve funcionar como contexto de aplicação.

## Quando não usar projeto

Não relacionar com projeto quando:

- o usuário disser que não quer;
- o usuário não informar projeto;
- a relação for forçada;
- o objetivo for estudar o assunto de forma geral.

## Pergunta curta alternativa

Quando a IA precisar ser breve, perguntar:

"Quer que eu relacione o conteúdo com algum projeto seu ou prefere um material totalmente genérico?"
